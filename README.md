# PlanOra
An iOS planner application that reminds you of to-do tasks and helps you remain focused 

# Key Features
Tasks tracking
Focus Mode
Mood UpLifter

## Tasks Tracking
Keeps a track of the tasks user added, and the ones they completed. Shows a progress circle reflecting the completion percentage

## Focus Mode
Allows user to set a timer of their own choice to specifically focus on a task

## Mood Uplifter
Gives user a motivational, uplifting quote based on the mood they tap on.
